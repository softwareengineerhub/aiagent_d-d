прямых «полностью бесплатных» видеогенераторов с глубокой интеграцией именно в Zapier Agents почти нет; есть сервисы с бесплатными или trial‑тарифами и Zapier‑интеграцией, которые можно использовать в связке с Agents.

Варианты, на которые стоит смотреть
Runway (через API + Zapier)

Runway можно подключать к Zapier через API‑endpoint и использовать в автоматизированных сценариях для генерации видео.

У Runway есть бесплатный план с ограниченными кредитами для видео; подойдёт для тест‑драйвов и прототипов.

Creatomate

Имеет официальную Zapier‑интеграцию для создания AI‑видео по шаблонам (текст, медиа, параметры приходят из Zapier).

Есть возможность использовать это в автоматизированных workflow, а pricing включает бесплатный/демо‑уровень для начала (детали нужно смотреть у них на сайте).

Last Video AI

Есть интеграция Last Video AI by Zapier, которую можно связать с AI by Zapier/Agents.

Указано, что доступен «Free tier available / Integrate for free», то есть есть бесплатный уровень доступа; детали лимитов — в самом сервисе.

Как использовать это с Zapier Agents
Agents генерирует сценарий, сегменты, промпты и метаданные.

Дальше:

либо сам агент вызывает action выбранного видео‑сервиса (если он есть в списке Tools для Agents),

либо ты делаешь классический Zap, который принимает данные от агента (Docs, Tables, Webhook) и дергает видеосервис.
